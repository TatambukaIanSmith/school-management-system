Six powerful new features have been added to enhance the School Registration System:

## 1. Student Photos 📸

### Features
- Upload student photos during registration
- Support for JPG, PNG formats
- Maximum file size: 2MB
- Image preview before saving
- Display photos in student table
- Photos included in print output

### How to Use
1. Click "Choose Photo" button in registration form
2. Select an image file from your computer
3. Preview appears immediately
4. Photo saves with student record
5. View photo in student list table

### Technical Details
- **Storage**: Base64 encoded in localStorage
- **Validation**: File type and size checked
- **Preview**: Real-time image preview
- **Fallback**: Default avatar icon if no photo

### Benefits
- Visual identification of students
- Professional appearance
- Useful for Security role
- Enhanced ID card printing

---

## 2. Age Calculator 🎂

### Features
- Automatically calculates age from date of birth
- Displays age in student table
- Updates dynamically
- Sort students by age
- Age shown in print output

### How It Works
- Calculates years from DOB to current date
- Accounts for month and day differences
- Shows "—" if DOB not provided
- Format: "15 yrs", "17 yrs", etc.

### Use Cases
- Quick age verification
- Age-based grouping
- Eligibility checking
- Statistical analysis

- Status dropdown in registration form
- Color-coded status badges
- Filter students by status
- Status shown in table and print
- Default status: Active

### How to Use

2. Status saves with student record
3. Can be changed when editing

**Filter by Status:**
1. Use "All Status" dropdown above table
2. Select desired status
3. Table updates instantly

### Benefits
- Track student lifecycle
- Easy reporting
- Quick filtering
- Clear visual indicators

## Troubleshooting

### Photo Upload Issues

**Problem**: Photo won't upload  
**Solution**: 
- Check file size (must be < 2MB)
- Verify file type (JPG, PNG only)
- Try different image

**Problem**: Photo looks distorted  
**Solution**:
- Use square or portrait photos
- Minimum 200x200px recommended
- Maximum 2000x2000px

### Filter Issues

**Problem**: No results after filtering  
**Solution**:
- Check if filters are too restrictive
