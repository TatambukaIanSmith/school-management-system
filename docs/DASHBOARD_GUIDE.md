# Dashboard Guide - School Registration System

## Overview

Each user role in the School Registration System has a customized dashboard designed to show the most relevant information and provide quick access to common tasks.

## Dashboard Features by Role

### 🔐 Administrator Dashboard

**Purpose**: Complete system oversight and management

**Statistics Displayed**:
- Total Students
- Male Students
- Female Students  
- Number of Classes

**Quick Actions**:
- Add Student
- View All Students
- Export Data
- System Stats

**Data Visualizations**:
- Class Distribution (bar chart showing students per class)
- Recent Activity (timeline of recent registrations and updates)

**Best For**: School principals, system administrators who need full visibility

---

### 📋 Registrar Dashboard

**Purpose**: Focus on registration and record management

**Statistics Displayed**:
- Total Registered Students
- Registrations This Month
- Pending Updates
- Number of Classes

**Quick Actions**:
- New Registration
- View Records
- Export CSV
- Search Student

**Data Visualizations**:
- Recent Registrations (list of newly registered students)

**Best For**: Admissions officers, registrars managing student enrollment

---

### 👨‍🏫 Teacher Dashboard

**Purpose**: Classroom-focused student information

**Statistics Displayed**:
- Total Students
- Number of Classes
- Male/Female Ratio

**Quick Actions**:
- View Students
- Update Records
- Export List

**Data Visualizations**:
- Class Overview (distribution by class)
- Student List (recent students)

**Best For**: Teachers who need to view and update student information

---

### 💚 Counselor Dashboard

**Purpose**: Student support and guidance

**Statistics Displayed**:
- Total Students
- Number of Classes
- Gender Ratio Percentage

**Quick Actions**:
- View Students
- Update Info
- Export Data

**Data Visualizations**:
- Students by Class
- Student Directory

**Best For**: Guidance counselors, student support staff

---

### 👥 Staff Dashboard

**Purpose**: Basic student information access

**Statistics Displayed**:
- Total Students
- Male Students
- Female Students
- Number of Classes

**Quick Actions**:
- View Students
- Search

**Data Visualizations**:
- Student Directory (list view)

**Best For**: Administrative staff, librarians, nurses with view-only needs

---

### 🔒 Security Dashboard

**Purpose**: Student identification and verification

**Statistics Displayed**:
- Total Students
- Male Students
- Female Students

**Quick Actions**:
- View Students
- Search by ID

**Data Visualizations**:
- Student Identification List (comprehensive list for verification)

**Best For**: Security personnel who need to verify student identities

## Navigation

### Between Pages
- **Dashboard → Students**: Click "Students" in the top navigation
- **Students → Dashboard**: Click "Dashboard" in the top navigation

### Quick Actions
All dashboard quick action buttons link directly to relevant pages or functions:
- Buttons with page links navigate immediately
- Export buttons trigger CSV download
- Search buttons navigate to student page with search focused

## Dashboard Elements

### Statistics Cards
- Large, easy-to-read numbers
- Color-coded by category
- Hover effect for interactivity
- Icon representing the metric

### Quick Action Buttons
- Icon + label for clarity
- Hover animation for feedback
- Direct links to common tasks
- Disabled for unauthorized actions

### Data Visualizations
- **Bar Charts**: Show distribution with percentage
- **Lists**: Display recent or relevant students
- **Activity Feed**: Show recent system activity (admin/registrar)

## Responsive Design

### Desktop (1024px+)
- Full grid layout with 4 columns for stats
- Side-by-side visualizations
- All quick actions visible

### Tablet (768px - 1023px)
- 2-3 column grid for stats
- Stacked visualizations
- Compact quick actions

### Mobile (< 768px)
- Single column layout
- Stacked statistics cards
- 2-column quick actions
- Scrollable lists

## Tips for Each Role

### For Administrators:
- Check the Recent Activity feed daily
- Monitor class distribution for balance
- Use System Stats for reporting

### For Registrars:
- Track "This Month" registrations
- Use Quick Registration for efficiency
- Export regularly for backups

### For Teachers:
- Review Class Overview before term starts
- Update student records as needed
- Export class lists for offline use

### For Counselors:
- Use Student Directory for quick lookup
- Monitor gender distribution for programs
- Update student info during sessions

### For Staff:
- Use Search for quick student lookup
- Bookmark frequently accessed students
- View directory for contact information

### For Security:
- Use Search by ID for verification
- Keep identification list accessible
- Report discrepancies to administrators

## Keyboard Shortcuts

- `Ctrl/Cmd + K`: Focus search (when on Students page)
- `Esc`: Close modals
- `Tab`: Navigate between elements

## Dark Mode

Toggle dark mode using the switch in the top-right corner:
- ☀️ Light Mode: Better for bright environments
- 🌙 Dark Mode: Easier on eyes in low light

Preference is saved and persists across sessions.
