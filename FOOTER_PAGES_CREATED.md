# Footer Support Pages Created

## Overview
Created four comprehensive support pages for the footer links: Help Center, Documentation, Contact Us, and Privacy Policy.

### 1. Help Center (help-center.html)
**Purpose:** Provide quick answers to common questions and guide users to resources

**Features:**
- Search box for finding help topics

  - Student Management (registration, editing, CSV import)
  - Exam System (creating exams, grading, results)
  - Account & Settings (password, profile, dark mode)
- "Contact Support" call-to-action button
- Clean, organized layout with glassmorphism design

**Content Sections:**
- How to create an account
- First time login guide
- Understanding user roles
- Student registration process
- Exam creation and grading
- Profile management
- And more...

**Purpose:** Complete technical guide for using the system

**Features:**
- Sticky sidebar navigation for easy section jumping
- Comprehensive sections:
  - Introduction & Key Features
  - User Roles (Administrator, Registrar, Teacher, Secretary, Bursar, Student)
  - Student Management (registration, editing, CSV import)
  - Exam System (creation, grading)
  - Profile Management (Chrome-style switching)
  - Data Export (CSV functionality)
- Color-coded role descriptions
- Step-by-step instructions
- Professional documentation layout

**Content Highlights:**
- Detailed explanation of all user roles
- Complete workflows for common tasks
- Technical details about features
- Best practices and tips

**Purpose:** Allow users to get in touch with support team

**Features:**
- Contact form with fields:

  - Email Address

  - Message

- Link to Help Center for quick answers
- Responsive grid layout

**Contact Methods:**

- Support hours clearly displayed

**Purpose:** Explain data collection, usage, and user rights

  - Data Sharing and Disclosure
