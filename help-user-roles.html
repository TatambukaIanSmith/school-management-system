Understanding User Roles

Learn about different roles and their permissions

Administrator

Full system access with all permissions

  • Manage all users and students
  • Create, edit, and delete records
  • Access all system features
  • Export data

Registrar

Manage student registrations and records

  • Register new students
  • Edit student information
  • Delete student records
  • Export student data

Teacher

Create exams and grade submissions

  • View student information
  • Create and manage exams
  • Grade student submissions
  • Export data

Secretary

Manage records and documentation

  • Create and edit records
  • View all student data
  • Export documentation

Bursar

Access financial records and student data

  • View student records
  • Edit financial information
  • Export data

Student

Take exams and view results

  • View available exams
  • Submit exam answers
  • View grades and results
  • Manage profile
