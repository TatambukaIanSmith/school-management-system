## Data Flow

```
┌─────────────────────────────────────────────────────────────┐
│                      USER INTERACTION                        │
└────────────────────────┬────────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────────┐
│                    EVENT HANDLER (Page)                      │
│                                                              │
│  function handleLogin(email, password) {                    │
│    // Validation                                            │
│    // Call service                                          │
│  }                                                           │
└────────────────────────┬────────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────────┐
│                    SERVICE LAYER                             │
│                                                              │
│  export function login(email, password) {                   │
│    const users = getUsers();                                │
│    const user = users.find(...);                            │
│    // Validate                                              │
│    return createSession(user);                              │
│  }                                                           │
└────────────────────────┬────────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────────┐
│                    DATA STORAGE                              │
│                                                              │
│  localStorage.getItem('srs_users')                          │
│  localStorage.setItem('srs_session', ...)                   │
└────────────────────────┬────────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────────┐
│                    STATE UPDATE                              │
│                                                              │
│  setCurrentUser(session)                                    │
│  setPermissions(getUserPermissions(role))                   │
└────────────────────────┬────────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────────┐
│                    NAVIGATION                                │
│                                                              │
│  window.location.hash = '#/dashboard'                       │
└────────────────────────┬────────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────────┐
│                    ROUTER                                    │
│                                                              │
│  handleRoute() → Load DashboardPage                         │
└────────────────────────┬────────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────────┐
│                    RENDER PAGE                               │
│                                                              │
│  renderPage(DashboardPage())                                │
│  → Inject HTML into <div id="app">                          │
└─────────────────────────────────────────────────────────────┘
```
